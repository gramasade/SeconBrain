# 🧠 Second Brain — Ade Rizki

Second Brain pribadi — sistem pengetahuan terstruktur berbasis **Obsidian**.

## 📂 Struktur Vault

```
📁 00 - Meta/           → Config, templates, attachments
📁 01 - Home/           → Homepage / Dashboard
📁 02 - Daily Notes/    → Catatan harian (YYYY-MM-DD.md)
📁 03 - Projects/       → Proyek aktif
📁 04 - Areas/          → Tanggung jawab & peran berkelanjutan
📁 05 - Resources/      → Pengetahuan referensi & interest
📁 06 - Archives/       → Proyek/area yang sudah selesai
📁 07 - MOCs/           → Map of Content (hub tematik)
📁 08 - Inbox/          → Quick capture / temporary
```

## 🔗 Cara Koneksi

| Metode | Cara |
|--------|------|
| **Links** | `[[nama note]]` antar note |
| **Tags** | `#tag` di frontmatter atau inline |
| **MOCs** | Map of Content sebagai hub tematik |
| **Dataview** | Query otomatis di Home & MOC |

## 🚀 Quick Start

1. Buka Obsidian → **Open folder as vault** → pilih folder ini
2. Klik **Trust author & enable plugins** jika diminta
3. Buka `01 - Home/Home.md` dan mulai dari sana
4. Gunakan `Ctrl-Shift-D` untuk Daily Note hari ini
5. Capture ide cepat di `08 - Inbox/Inbox.md`

## ⚡ Essential Plugins (terdaftar, klik install saat buka)

- **Dataview** — query notes kayak database
- **Calendar** — kalender visual untuk daily notes
- **Kanban** — board untuk项目管理
- **Templater** — template advanced
- **Quick Add** — quick capture
- **Tasks** — task management
- **Tag Wrangler** — manage tags
